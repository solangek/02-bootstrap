A List of Computer Programming Languages

  • Java, a compiled, object-oriented, strongly-typed, language. Java's strengths include the ability to write code once, and then run anywhere with a JVM (Java virtual machine) installed. Java is used for enterprise applications, web applications, and Android mobile apps.
  • C#, a compiled, object-oriented language, created by Microsoft, and part of the .NET framework. It is used to create applications for the Windows platform, and in conjunction with ASP to create web applications.
  • Python is an open-source multi-paradigm language, supporting many styles of programming. It is widely used in many applications, including scripting, web applications, and data science.
  • Swift is a newer language created by Apple for use developing applications for the Mac OS-X platform, as well as for iOS mobile apps.
  • The initial prototype of JavaScript was created in 10 days in 1995. It is currently eating the world.
