Common HTML Elements in Bootstrap

Some HTML tags

This is a list of some common HTML tags on a page styled with Bootstrap 4. There is no other CSS on this page, other than that provided by Boostrap.

Text Emphasis

Italic and Bold <i> and <b> are used to display text in an italic font, such as for names of movies like Back to the Future III. These are physical properties of the text. You might also want to use bold text that is important.

Emphasised and Strong Text <em> and <string> There are also tags for emphasized text and for strong text which have slightly different meanings to italic <i> and bold <b> tags, but display the same in most browsers. These tags define logical properties and should be used instead of physical properties (tags above).

Escape characters To display special HTML characters on a page, such as > and < and &, you need to use escape sequences. So to display & you need to type "&amp" in your HTML, for > and < you need to type "&gt" and "&lt"

Formatting Blocks of Text

Preformatted text The <pre> tag is for pre-formatted text. It displays in a monospace font, and unlike p tags, lines of text that are wider than the browser window do not wrap,
  spaces        and returns
    and tabs          in the HTML source are preserved and displayed in the browser. This may be useful for code.

Code blocks There's also a <code> tag. Unlike pre, this mostly changes the font, does not preserve tabs, and wraps. You may need to do some extra work in CSS to display code nicely. console.log("hello world!");

Block quotes <blockquote> is another way to display text in a block; by default this text is indented, but in Bootstrap the text and it's purpose is to display quotes.

Organizing Sections

Span <span> tags are mostly used for applying styles to text within a paragraph. There's a span in this paragraph but you'll only see it if you look at the source, or apply a visible style to span tags.

Div <div> tags are for organizing content into sections. It's common to add a class attribute to label your sections.

Organizing Information

Tables use many tags. <table>, <tr>, <th>, <td>... There are some example tables at W3Schools.

It's very common to use extra CSS to apply styles to tables, for example, borders, or background colors to cells, since the default version is very basic.

Images

Images are important to brand and personalize a website. To add an image, use the >img< tag, and remember to specify the image's location. This can either be the path to the image, relative to where you page is saved OR a URL.

Download and save various sample images from a public domain site like pixabay.

An image saved in the same directory as your web page can be displayed like so:

<img src="kittens.jpg">

And an image resource from another website can be displayed like this:

<img src="https://placebear.com/200/400">

Lists

Ordered and unordered lists There are two types of lists, ordered, with numbers; and unordered, with bullet points.

Use the <ol> tag to create an ordered list, then the <li> tag to define individual numbered list elements

  1. First of all...
  2. And then you should....
  3. And finally...

Use the <ul> tag to create a list, then the <li> tag to define individual bullet-point list elements.
