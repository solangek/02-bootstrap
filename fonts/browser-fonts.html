Example page using browser fonts

This page may look different between browsers. The browser will attempt to use the font specified, but if it doesn't know that font, then the text will be the default browser font. So, it's useful to provide a list of fonts including a web-safe font that all browsers should know.

So for this style, 

  
    body {
      font-family: "Lucida Console", Monaco, monospace;
    }

The browser will try to use the font Lucida Console first. If it can't find that font, it will try Monaco, and it it can't find that font, it will use a font in the serif family. There are a set of font-families that all browsers should know, for examples serif, san-serif, monospace.
