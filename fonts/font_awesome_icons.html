Icons can be different sizes, too! Add the fa-2x or fa-3x ... class to set the size.

What does the aria-hidden attribute do? From FontAwesome's website,

Note: to improve web accessibility, we recommend using aria-hidden="true" to hide icons used purely for decoration.

While some icons are decorative, other icons have meaning on a page. For example, you might use a plane icon for the user to click to check on a flight arrival. To make this accessible, add a title to the icon, which will be shown as a tool tip, and a span for screen readers.

Hover over the plane icon, you'll see a tool tip.
